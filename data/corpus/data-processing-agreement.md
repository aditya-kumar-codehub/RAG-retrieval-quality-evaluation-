# Data Processing Agreement (DPA)

Northwind Cloud is SOC 2 Type II certified and offers a Data Processing Agreement (DPA) to support customers' GDPR and CCPA compliance obligations. The DPA is available to customers on any paid tier (Starter, Pro, or Enterprise) upon request through the dashboard's legal settings page; it is not available to Free tier accounts, who are considered to be in an evaluation period.

The DPA governs how Northwind Cloud processes personal data on the customer's behalf, including a list of approved subprocessors (currently: AWS for infrastructure hosting, and Sendgrid for transactional email delivery). Any change to the subprocessor list is announced to DPA-holding customers at least 30 days in advance via email.

Data subject deletion requests under GDPR are processed within 30 days, which takes precedence over the standard data retention windows described in `data-retention.md`. Data subject access requests (a user asking what data is held about them) are fulfilled within 30 days as well, and can be satisfied in part using the standard data export functionality described in `data-export.md`.

Enterprise customers with specific regional data residency requirements should discuss options with their account manager; standard hosting is in US-based AWS regions.
