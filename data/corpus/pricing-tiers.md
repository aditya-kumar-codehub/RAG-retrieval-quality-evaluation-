# Pricing Tiers

Northwind Cloud offers four pricing tiers:

**Free** — $0/month. Includes 1,000 API calls per month, community forum support only, and 30-day data retention. Rate limited to 60 requests per minute.

**Starter** — $49/month. Includes 50,000 API calls per month, email support with a 24-hour response SLA, and 30-day data retention. Rate limited to 300 requests per minute. Overage is billed at $2 per additional 1,000 calls.

**Pro** — $299/month. Includes 500,000 API calls per month, live chat support with a 4-hour response SLA, 90-day data retention, and access to custom domains (see `custom-domains.md`). Rate limited to 1,500 requests per minute. SSO/SAML is not included at this tier. Overage is billed at $1.50 per additional 1,000 calls.

**Enterprise** — custom pricing, negotiated per contract. Includes unlimited API calls (subject to a negotiated fair-use ceiling), dedicated support with a 1-hour response SLA, configurable data retention up to 7 years, SSO/SAML (see `sso-saml.md`), audit logs (see `audit-logs.md`), and a custom rate limit (see `rate-limits.md`).

All tiers can be changed at any time from the billing dashboard; upgrades take effect immediately and downgrades take effect at the start of the next billing cycle. Billing mechanics, including invoicing and dunning for failed payments, are covered in `billing-invoicing.md`. Support response-time commitments by tier are detailed further in `customer-support-sla.md`.
